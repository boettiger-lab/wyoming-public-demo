# geo-agent: Public GYE

[![DOI](https://zenodo.org/badge/1173817623.svg)](https://doi.org/10.5281/zenodo.20674178)

Configuration for **Wyoming Wildlife & Lands**, a wildlife-habitat and public-lands decision-support map app with a natural-language data agent. **Live:** <https://wyoming-public-demo.nrp-nautilus.io>

The map, chat, and agent code live in the [geo-agent](https://github.com/boettiger-lab/geo-agent) library (loaded from CDN); this repo only carries the configuration — `index.html`, `layers-input.json`, `system-prompt.md`, and `k8s/`. See [AGENTS.md](AGENTS.md) to configure or deploy.
